다른 언어와 비교

→ Python: json.dumps / json.loads

→ JS: JSON.stringify / JSON.parse

Go JSON 핵심 특징

→ 구조체 태그 `json:"name"` 매핑 ✅

→ Marshal/Unmarshal 한 쌍이면 끝 🔧

→ omitempty로 빈 값 자동 제외 🎯

→ 중첩 구조체도 자동 처리! 📦

Go 1분 정리 EP.15

구조체 → JSON (Marshal)

type User struct {
    Name  string `json:"name"`
    Email string `json:"email"`
    Age   int    `json:"age"`
}

func main() {
    user := User{
        Name: "고퍼", Email: "go@go.dev",
        Age: 10,
    }
    // 구조체 → JSON 바이트
    data, err := json.Marshal(user)
    if err != nil { log.Fatal(err) }
    fmt.Println(string(data))
    // {"name":"고퍼","email":"go@go.dev","age":10}

⭐ json.Marshal = Go 객체를 JSON으로 변환!

Go 1분 정리 EP.15

JSON → 구조체 (Unmarshal)

jsonStr := `{
    "name": "고퍼",
    "email": "go@go.dev",
    "age": 10
}`

var user User
err := json.Unmarshal(
    []byte(jsonStr), &user,
)
if err != nil {
    log.Fatal(err)
}
fmt.Println(user.Name)  // 고퍼
fmt.Println(user.Email) // go@go.dev
fmt.Println(user.Age)   // 10

💡 Unmarshal 두 번째 인자는 반드시 포인터(&)!

Go 1분 정리 EP.15

구조체 태그 활용

type Product struct {
    ID    int    `json:"id"`
    Name  string `json:"name"`
    // omitempty: 빈 값이면 JSON에서 제외
    Desc  string `json:"desc,omitempty"`
    // -: JSON 변환에서 완전 제외
    Secret string `json:"-"`
}
p := Product{ID: 1, Name: "Go북"}
data, _ := json.Marshal(p)
fmt.Println(string(data))
// {"id":1,"name":"Go북"}
// desc 빈값 → 생략됨!
// Secret → 완전 제외!

⭐ omitempty = 빈 값 생략, "-" = 완전 제외!

Go 1분 정리 EP.15

중첩 구조체 JSON

type Address struct {
    City string `json:"city"`
    Zip  string `json:"zip"`
}
type Person struct {
    Name    string  `json:"name"`
    Address Address `json:"address"`
}
p := Person{
    Name: "고퍼",
    Address: Address{
        City: "서울", Zip: "06000",
    },
}
data, _ := json.MarshalIndent(
    p, "", "  ",
)
fmt.Println(string(data))

💡 MarshalIndent = 예쁘게 들여쓰기된 JSON!

Go 1분 정리 EP.15

map과 동적 JSON 처리

// 구조체 없이 map으로 처리
jsonStr := `{"name":"Go","ver":1.21}`

var m map[string]any
json.Unmarshal(
    []byte(jsonStr), &m,
)
fmt.Println(m["name"]) // Go

// map → JSON
data := map[string]any{
    "status": "ok",
    "count":  42,
}
result, _ := json.Marshal(data)
fmt.Println(string(result))
// {"count":42,"status":"ok"}

💡 구조 모를 때 map[string]any 활용!

Go 1분 정리 EP.15

실전: API 응답 파싱 패턴

// API 공통 응답 구조
type ApiResponse struct {
    Code int         `json:"code"`
    Msg  string      `json:"msg"`
    Data json.RawMessage `json:"data"`
}
type UserData struct {
    ID   int    `json:"id"`
    Name string `json:"name"`
}
// 2단계 파싱 패턴
var resp ApiResponse
json.Unmarshal(body, &resp)
if resp.Code == 200 {
    var user UserData
    json.Unmarshal(resp.Data, &user)
}

⭐ json.RawMessage로 지연 파싱 = 실무 필수!
