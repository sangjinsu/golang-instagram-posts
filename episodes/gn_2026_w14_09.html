왜 중요한가? 💡

📊 마이크로서비스 PR 1개당 평균 11개 외부 의존성 — 11개의 네트워크 장애 지점

⚡ API 에뮬레이션 도입 팀: CI 시간 20~60% 단축 실증 데이터

🔴 외부 API 의존 테스트: 팀의 5%가 코드 변경 없이도 매 실행마다 CI 실패 경험

🚀 핀테크 팀 실제 사례: CI 신뢰도 92% → 99.2%, 파이프라인 2분 11초 단축

외부 API 없이 테스트 → 더 빠르고 더 안정적인 CI 💪

핵심 기능 살펴보기 🔍

🔧 7개 서비스 지원

Vercel·GitHub·Google·Slack·Apple·Microsoft·AWS — 각각 고유 포트(4000~4006) 할당

🔐 OAuth 완전 지원

OAuth 2.0·OIDC·GitHub App JWT 인증·Webhook 전달까지 완벽 재현

💾 완전 Stateful

생성·수정·삭제가 메모리에 지속 — Mock과 달리 실제 API처럼 상태 유지

🧪 테스트 프레임워크 통합

Vitest/Jest에서 reset()·close()로 테스트 간 격리 보장, 프로그래밍 방식 제어 가능
